# Graph Script Editor

Web-приложение для визуального построения графов и запуска скриптов на них.

Построено в режиме vibe coding совместно с Claude Code.

## Что это

Интерактивный редактор графов с встроенным редактором кода. Рисуешь граф мышкой — пишешь функцию на JavaScript — жмёшь «Запуск» — получаешь результат.

Подходит для экспериментов с алгоритмами на графах, учёбы или просто чтобы поиграться.

## Возможности

- Добавление и удаление вершин (строгая нумерация 1..n)
- Создание направленных рёбер кликами по вершинам
- Перетаскивание вершин по холсту, зум и панорамирование
- Рёбра с физикой нитей — пружинисто колеблются при перетаскивании вершин
- Перетаскиваемые разделители панелей (граф/код и редактор/вывод)
- Включаемая/отключаемая сетка на холсте
- Редактор кода с подсветкой синтаксиса (CodeMirror)
- Выполнение пользовательских скриптов в песочнице (Web Worker, таймаут 5 сек)

## Запуск

Сборка не нужна. Зависимости подгружаются через CDN.

```bash
# любой HTTP-сервер, например:
python -m http.server 8080

# или
npx serve .
```

Открыть `http://localhost:8080`.

## Как писать скрипты

В редакторе нужно определить функцию `solve` с фиксированной сигнатурой:

```javascript
function solve(n, m, edges) {
  // n     — количество вершин
  // m     — количество рёбер
  // edges — массив пар [u, v], нумерация с 1
  //
  // Возвращайте что угодно: число, строку, массив, объект.

  return n + ' вершин, ' + m + ' рёбер';
}
```

Результат отображается в панели вывода под редактором.

## Стек

- [Cytoscape.js](https://js.cytoscape.org/) — визуализация и взаимодействие с графом
- [CodeMirror 5](https://codemirror.net/5/) — редактор кода
- Vanilla JS, без фреймворков и сборщиков
